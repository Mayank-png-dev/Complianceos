# ComplianceOS 
A backend system to track GST compliance for businesses and Chartered Accountants.

## Features (MVP)
- GSTIN lookup (mock + API ready)
- Structured backend architecture
- MongoDB Atlas integration
- Scalable service-based design

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Axios

## Project Structure
backend/
├── config/
├── models/
├── routes/
├── controllers/
├── services/

git clone https://github.com/your-username/complianceos.git
